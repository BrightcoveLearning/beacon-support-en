Prerequisites

In Beacon Studio:

  • Navigate to the Cloud Playout module. Then, fill the schedule with content and make sure it is in a start/running status.

  • It is necessary to have the Channel ID.

    • Navigate to the Live Module for the channel. You will need:

    • Video ID (via standard embed code).
    • Ad Config ID .
      • Note: You can find this information by clicking the Live streaming.

In Beacon Classic:

  1. Login to Beacon Classic.
  2. Navigate to Advertisement.
  3. Click Add New Configuration.
  4. Create Advertisement as SSAI with Live Ad Config ID.

    Click Create New Advertisement.

  5. Navigate to Channels.
  6. Click the Channel. It's sent to Basic Data. Localize CloudPlayOut Channel ID and fill it with the Channel ID. Click Update Channel.
  7. Navigate to streams and add the Stream Video ID. In Stream Type select Live.

    Click Add new stream.

  8. Navigate to Rights & Scheduling. Set the AVOD and Advertisement Configuration created.
  9. Click to configuration and fetch EPG.
  10. Go back to Channels, click EPG and click action to add the items.

Replacing an Existing Channel

In Beacon Studio:

    • Navigate to the Cloud Playout module. Then, fill the schedule with content and make sure it is in a start/running status.

    • It is necessary to have the Channel ID and Video ID (BC Live).

      • In Beacon Classic:

      1. Login to Beacon Classic.
      2. Navigate to Channel, then clock Basic Data.
      3. Change the CloudPlayOut Channel ID.
      4. Click Streams and change the Video ID.
      5. Go to the Advertisement Configuration and change to the new ID.
      6. Refresh the page in Beacon.
        7. Note: Without refreshing, viewers watching the current program will continue with the channel configuration as loaded on the page.
      7. Click to configuration and fetch EPG.
      8. Go back to Channels, click EPG and click action to add the items.
        9. Note: A large number of channels can take time to load the EPG configurations.
