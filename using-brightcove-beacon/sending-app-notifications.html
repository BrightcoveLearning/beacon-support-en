Introductions

For iOS and Android apps you can send notifications using Google Firebase. This document will walk you through the process. The configuration of the iOS and/or Android apps to enable this functionality is done be Brightcove.

Sending a notification

Configure the notification.

  1. Navigate to the Firebase Console and select your project.
  2. In the left navigation select:
    • Engage
    • Cloud Messaging
  1. Click Send your first message.
  2. Supply the notification title and text. Note that you can see a preview on the bottom right of the screen.
  3. Add an optional notification image.
  4. Add an optional notification name.
  5. If configured, you can test the message.
  6. Click Next.

Define the target

When you define the target you configure which apps should receive the notification and also define any further restrictions on whom should receive the notification.

  1. Select the app or apps to which you wish to send the notification.
  2. Add any restrictions for each app to which you are sendiung the notification.
  3. Click Next.

Schedule when the notification will be sent.

You have the option to send the notification immediately or have it sent repeatedly.

Add options for the video asset to which the notification will link.
