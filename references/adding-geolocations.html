Introduction

In Beacon Classic, you can define locations to include or exclude where your videos can be viewed.

Adding a new location

Follow these steps to add a new location:

  1. Log in to Beacon Classic.
  2. Navigate to Locations by following these steps:
  3. Click Settings in the top navigation.
  4. Click Locations in the left navigation.
  5. To add a new location, click Add new location.
  6. Now you have to choose:
    1. Location name.
    2. Country.
      3. Note: the Location name could be different from the Country, it does not have to be identical.
  7. Notice that you can choose other locales:
    • City
    • Zip
    • Fips
    • Dst
    • Type
    • State
    • Zone
    • Ac
    • Tz
    • Ma
    • DMAs

    Adding Geo-restrictions on videos

    To control which locations a video can be viewed, you can add geo-restrictions for the video. Follow these steps to add a new geo-restriction:

    1. In the top navigation, click on the category that you want to use. In this case, it would be Movies.
    2. Select the video you wish to use.
    3. Click Rights & Scheduling.
    4. Once you have created a location, you have two options:
      • Click Permitted locations - To include certain regions
      • Click Denied locations - To exclude certain regions
