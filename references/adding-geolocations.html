Introduction

App users use geo-locations on Beacon Classic when they want to set up locations to include or exclude certain regions to be able to see videos.

Adding a new location

Follow these steps to add a new location:

  1. Log in to Beacon Classic.
  2. Navigate to Locations by following these steps:
    1. Click Settings in the top navigation.
    2. Click Locations in the left navigation.
  3. To add a new locations, click Add new location.
  4. Now you have to choose: a location name and a country. Click save. Notice that you can choose other locales like Country, City, Zip, Fips, Dst, Type, State, Zone, Ac, Tz, Ma, DMAs, but it is not required.
    5. Note: the Location name could be different from the Country, it does not have to be identical.

Add Geo-restrictions on videos

Follow these steps to add a new geo-restriction to make a video unavailable to watch in specific locations.

  1. Click on the top navigation to open the category you want to use. In this case, it would be Movies.
  2. Select the video you wish to use.
  3. Click Rights & Scheduling
  4. Once you have created a location, you have two options:
    • Click Permitted locations - To include certain regions.
    • Click Denied locations - To exclude certain regions.
