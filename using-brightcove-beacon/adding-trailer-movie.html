Introducing trailers

A trailer is a short clip associated with a video meant to entice your audience and build excitement for your content. Trailers can be added to video content in both Beacon Classic and Video Cloud Studio.

Within Beacon Classic, you can add trailer to the following:

  • Movies
  • Series
  • Seasons

You add trailers to Video Cloud videos in Video Cloud Studio. These videos when ingested into Brightcove Beacon will be movies. The process of adding trailers in Video Cloud Studio requires the Video Cloud custom field beacon.trailer.id. For information and directions on how to add a custom field to your Video Cloud account see the Creating Video Cloud Custom Fields document. Your custom field should be defined as follows:
Trailer Custom Field
Display Name Internal Name
beacon.trailer.id beacon_trailer_id

Note that if a video is used as a trailer, it will be regarded as a Movie or an Episode within Beacon.

Adding a trailer to a Video Cloud video

You will add the trailer to a Video Cloud video, and when ingested into Brightcove Beacon it becomes a movie. Follow the steps below:

  1. Upload the trailer video to your Video Cloud account.
  2. Copy the ID for the trailer video:
  3. From the Media module, click the video title to which you are adding a trailer to open the properties page. In the Custom Fields section, paste the trailer video ID into the beacon.trailer.id custom field:
  4. Click Save.

To see your results immediately in your Brightcove Beacon apps, clear the cache in Beacon Classic, as explained in the Using the Tools document.

Using Beacon Classic to add a trailer

In Beacon Classic you can add trailers to movies, series and seasons. The process is very similar for all three entities. The screenshots shown below are for adding a trailer to a series, but the process is nearly identical for movies and seasons.

  1. Select the series to which you wish to add a trailer.
  2. Click the Streams tab.
  3. Be sure the Provider is Brightcove().
  4. Click Add new stream.
  5. Add the Video Cloud ID for the trailer in the Stream Video ID field.
  6. Select the Stream Type to be Trailer.
  7. Click Update Series

To see your results immediately in your Brightcove Beacon apps, clear the cache in Beacon Classic, as explained in the Using the Tools document.
